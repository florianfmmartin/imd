Fibonacci

Hi I'll implement fibonacci in deno

Recursrively

const fib_rec = (n) => n < 2 ? n : fib_rec(n-1) + fib_rec(n-2);

With cache

let cache_fib = {};
const fib = (n) => {
    try { return cache_fib[n] }
    catch {
        const r = fib(n-1) + fib(n-2)
        cache_fib[n] = r
        return r
    }
}
